The time in minutes to wait before aborting running builds.
Set to a negative value to wait indefinitely.
The format is '<integer>' in minutes, '<integer><unit>' or a combination of '<integer><unit>', where <unit> is one of 'm' (minutes), 'h' (hours) or 'd' (days). Example: '1d 4h 30m' means one day, four hours and 30 minutes which evaluates to 1710 minutes. 1710 is then the value that will be displayed later.
